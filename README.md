# Life Science Chatbot POC

A proof-of-concept AI chatbot built for the life science domain. Enables natural language querying of scientific and clinical data using Large Language Models (LLMs), making complex life science information accessible through simple conversation.

## Live Demo

👉 [Try the Chatbot](https://lifescience-chatbot-poc-jbbcmgikfkugfwpdhkumsc.streamlit.app/)

---

## What It Does

- Ask questions about life science topics in plain English
- Get intelligent responses powered by LLM
- Domain-specific knowledge for clinical and scientific data
- Simple and clean chat interface built with Streamlit
