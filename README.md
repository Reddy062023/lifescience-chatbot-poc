# Life Science Chatbot POC

A proof-of-concept AI chatbot built for the life science domain. Enables natural language querying of scientific and clinical data using LLMs.

## Live Demo

👉 [Try the Chatbot](https://lifescience-chatbot-poc-jbbcmgikfkugfwpdhkumsc.streamlit.app/)

## Tech Stack

| Layer | Technology |
|---|---|
| Language | Python |
| UI | Streamlit |
| AI / LLM | OpenAI GPT |
| Domain | Life Science / Clinical |

## Run Locally

```bash
pip install -r requirements.txt
streamlit run chatbot_app.py
```
